# new 的实现过程

在调用 new 的过程中会发生以上四件事情：

  • 新生成了一个对象
  • 链接到原型
  • 绑定 this
  • 返回新对象

以下是对实现的分析：

  • 创建一个空对象
  • 获取构造函数
  • 设置空对象的原型
  • 绑定 this 并执行构造函数
  • 确保返回值为对象
