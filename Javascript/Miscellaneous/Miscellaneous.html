# 深拷贝

可以通过 JSON.parse(JSON.stringify(object)) 来解决。

  let a = {
    age: 1,
    jobs: {
      first: 'FE'
    }
  }
  let b = JSON.parse(JSON.stringify(a))
  a.jobs.first = 'native'
  console.log(b.jobs.first) // FE
