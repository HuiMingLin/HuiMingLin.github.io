排序分类： 1、函数调用所需的栈和固定数目的实例变量之外无需额外内存的原地排序算法 2、需要额外内存空间来存储另一份数组副本的其他排序算法

倒置：数组中的两个顺序颠倒的元素 部分有序的：数组中倒置的数量小于数组大小的某个倍数

# 选择排序

  • 找到数组最小的那个元素
  • 将它和数组的第一个元素交换位置（如果第一个元素就是最小的元素那么它和自己交换）
  • 在剩下的元素中找到最小的元素，将它和数组的第二个元素交换位置
  • 如此反复，直到将整个数组排序
public static void sort(Comparable[] a) {
  // 将 a[] 按升序排列
  int N = a.length;
  for (int i = 0; i < N; i++) {
      // 将 a[i] 和 a[i+1...N] 中最小的元素交换
      // 最小元素的索引 把第一个数当做最小的数
      int min = i;
      for (int j = i+1; j < N; j++) {
          if(less(a[j], a[min])) {
              min = j;
          }
      }
    exch(a, i, min);
  }
}

# 插入排序

对于一个有序的数组，会发现每个元素都已经在合适的位置之上，运行时间是线性的

将 a[i] 与 a[0] 和 a[i-1] 中比它小的所有元素依次有序地交换。 在索引i由左向右的变化的过程中，它左侧的元素总是有序的，所以当i到达数组的右端时排序就完成了

public static void sort(Comparable[] a) {
    // 将 a[] 按升序排列
    int N = a.length;
    // 第一个数确认是排序完成的
    for (int i = 1; i < N; i++) {
        // 如果后一个数比前一个数小，将后一个数往前移一位
        // 将 a[i] 插入到 a[i-1]、a[i-2]、a[i-3]... 之中
        for (int j = i; j > 0 && less(a[j], a[j-1]); j--) {
            exch(a, j, j-1);
        }
    }
}

# 希尔排序

交换不相邻的元素已对数组的局部进行排序， 使数组中任意间隔为h的元素都是有序的（有序h数组）

对于任意以1结尾的h序列，我们都能够将数组排序，算法实现以 1/2(3^k-1), 从 N/3 开始递减至1 成为递增序列 5

public static void sort(Comparable[] a) {
  // 将 a[] 按升序排列
  int N = a.length;
  int h = 1;
  while (h < N / 3) {
      h = 3 * h + 1; // 1 4 13 40 121
  }
  while (h >= 1) {
      // 将数组变为h有序
      for (int i = h; i < N; i++) {
          // 将 a[i] 插入到 a[i-h]、a[i-2*h]、a[i-3*h]... 之中
          for (int j = i; j >= h && less(a[j], a[j - h]); j -= h) {
              exch(a, j, j-h);
          }
      }
      h = h / 3;
  }
}

# 归并排序

先将前半部分排序，再将后半部分排序，然后在数组中移动元素而不需要使用额外的空间

# 原地归并的抽象方法

  • 左半边用尽（取右半边的元素）
  • 右半边 用尽（取左半边的元素）
  • 右半边的当前元素小于左半边的当前元素（取右半边的元素）
  • 右半边的当前元素大于等于左半边的当前元素（取左半边的元素）
public static void merge(Comparable[] a, int lo, int mid, int hi) {
    // 将 a[lo..mid] 和 a[mid+1..hi] 归并
    int i = lo, j = mid + 1;
    for (int k = lo; k < hi; k++) {
        aux[k] = a[k];
    }
    for (int k = lo; k <= hi; k++) {
        if(i > mid) {
            a[k] = aux[j++];
        } else if(j > hi) {
            a[k] = aux[i++];
        } else if(less(aux[j], aux[i])) {
            a[k] = aux[j++];
        } else {
            a[k] = aux[i++];
        }
    }
}

# 自顶向下的归并排序

# 优化
  • 对小规模子数组使用插入排序
  • 测试数组是否已经有序（如果a[mid] 小于等于 a[mid+1], 我们就认为数组已经是有序的并跳过 merge 方法）
  • 节省数组元素复制到用于归并的辅助数组所用的时间
    • 将数据从输入数组排序到辅助数组
    • 将数据从辅助数组排序到输入数组
public class Merge {
    private static Comparable[] aux;
    private static boolean less(Comparable v, Comparable w) {
        return v.compareTo(w) < 0;
    }

    // 原地归并的抽象方法
    public static void merge(Comparable[] a, int lo, int mid, int hi) {
        // 将 a[lo..mid] 和 a[mid+1..hi] 归并
        int i = lo, j = mid + 1;
        for (int k = lo; k < hi; k++) {
            aux[k] = a[k];
        }
        for (int k = lo; k <= hi; k++) {
            if(i > mid) {
                a[k] = aux[j++];
            } else if(j > hi) {
                a[k] = aux[i++];
            } else if(less(aux[j], aux[i])) {
                a[k] = aux[j++];
            } else {
                a[k] = aux[i++];
            }
        }
    }

    // 自顶向下的归并排序（从多到少）
    public static void sort(Comparable[] a) {
        aux = new Comparable[a.length];
        sort(a, 0, a.length - 1);
    }
    private static void sort(Comparable[] a, int lo, int hi) {
        // 将数组 a[lo...hi] 排序
        if(hi <= lo) return;
        int mid = lo + (hi - lo) / 2;
        sort(a, lo, mid); // 将左半边排序
        sort(a, mid + 1, hi); // 将右半边排序
        merge(a, lo, mid, hi); // 归并结果
    }
}

# 自底向上的归并排序

先归并那些微型数组，然后再成对归并得到的子数组，如此这般，知道我们将整个数组归并到一起

首先是两两归并，其次是四四归并，然后是八八归并，一直下去

public class MergeBU {
    private static Comparable[] aux;
    private static boolean less(Comparable v, Comparable w) {
        return v.compareTo(w) < 0;
    }

    // 原地归并的抽象方法
    public static void merge(Comparable[] a, int lo, int mid, int hi) {
        // 将 a[lo..mid] 和 a[mid+1..hi] 归并
        int i = lo, j = mid + 1;
        for (int k = lo; k < hi; k++) {
            aux[k] = a[k];
        }
        for (int k = lo; k <= hi; k++) {
            if(i > mid) {
                a[k] = aux[j++];
            } else if(j > hi) {
                a[k] = aux[i++];
            } else if(less(aux[j], aux[i])) {
                a[k] = aux[j++];
            } else {
                a[k] = aux[i++];
            }
        }
    }

    // 自顶向下的归并排序（从多到少）
    public static void sort(Comparable[] a) {
        int N = a.length;
        aux = new Comparable[N];
        for (int sz = 1; sz < N; sz = sz + sz) { // 子数组大小
            for (int lo = 0; lo < N - sz; lo += sz + sz) { // 子数组索引
                merge(a, lo, lo + sz - 1, Math.min(lo + sz + sz - 1, N - 1));
            }
        }
    }
}

# 快速排序

function quickSort(ary, left, right) {
    if(left >= right) {
        return ary;
    }

    var i = left,
        j = right,
        base = ary[left];

    while (i < j) {
        // 从右边起，寻找比基数小的数
        while (i<j && ary[j] >= base) {
            j--;
        }

        // 从左边起，寻找比基数大的数
        while (i<j && ary[i] <= base) {
            i++
        } 

        if (i<j) {
            var temp = ary[i];
            ary[i] = ary[j];
            ary[j] = temp;
        }
    }

    ary[left] = ary[i];
    ary[i] = base;

    quickSort(ary, left, i-1);
    quickSort(ary, i+1, right);

    return ary;
}

# 去重

  public static int removeDuplicates(int[] a) {
    if (a.length == 0) return 0;
    int j = 0;

    for (int i = 0; i < a.length; i++) {
      if (a[j] != a[i]) {
        a[++j] = a[i];
      }
    }
    return j + 1;
  }
