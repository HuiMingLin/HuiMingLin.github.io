# 命名规范

变量名大小写敏感，--header-color和--Header-Color是两个不同变量。

普通字符局限在只要是“数字[0-9]”“字母[a-zA-Z]”“下划线_”和“短横线-”这些组合，但是可以是中文，日文或者韩文

body {

  --深蓝: #369;

  background-color: var(--深蓝);

}

# 全局变量

:root伪类可以看做是一个全局作用域，在里面声明的变量，页面下面的所有选择器都可以拿到变量

:root {

  --color: blue;

}
.box{

  color: var(--color);

}

变量作用域同一个 CSS 变量，可以在多个选择器内声明。读取的时候，优先级最高的声明生效。这与 CSS 的"层叠"（cascade）规则是一致的

<style>

  :root { --color: blue; }

  div { --color: green; }

  #alert { --color: red; }

  * { color: var(--color); }

</style>

# var()函数可以使用第二个参数

表示变量的默认值。如果该变量不存在，就会使用这个默认值。

color: var(–color,#ccc);

# var()函数用在变量的声明

# 变量值类型

如果变量值是一个字符串，可以与其他字符串拼接。

--bar: 'hello';

--foo: var(--bar)' world';
1
2
3

如果变量值是数值，不能与数值单位直接连用。

.child {

    --size: 20;

    /*无效*/

    font-size: var(--size)px;

}
