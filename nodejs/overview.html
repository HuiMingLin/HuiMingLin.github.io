# 模块

exports = module.exports = xxx

require：用来引用模块

export：用来导出模块，包括标识符（identifier）和模块内容（contents），返回的是引用，同步加载

module.exports： 对外导出的对象只能有1个。 module.xxx： 对外导出的值可以是多个。

当多次引用一个模块时，该模块只会加载一次，其他情况下都在缓存中加载，不需要重新加载。

# npm(node package manager)

版本 v2 典型的树形依赖，层次深，依赖重叠，安装时间长，但相对更清晰 v3 扁平化依赖，将依赖移到了顶层，同一模块多版本依赖时需采用 v2 模式，下载速度变快 v5 自动记录依赖树，下载使用强校验，重写缓存系统等功能，新增 package-lock.json，用于记录和锁定依赖树的信息，与 yarn 类似

# nrm（node registry manager）

测速：nrm test 查看源的状态：nrm ls 切换源：nrm use xxx 增加源：nvm add yourcompany xxx
